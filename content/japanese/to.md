## 因果关系

1. 自然条件

```example
ボタンを押すと、店員が来る。#一按按钮，店员就会来。
```

2. 连续动作。

```example
ボタンを押すと、店員が来た。#一按按钮，店员就来了。
ご飯を食べると、出かけた。#一吃完饭就出去了。
```

3. 必然发生的事情。

```example
ご飯を食べると眠くなる。#一吃饭就会变困。
```

## 并列关系

```example
猫と犬。#猫和狗。
```

## 对象

1. 合作对象

```example
彼女と喧嘩する。#和女友吵架。
```

2. 比较对象

```example
彼と同じ年です。#和他一样大。
```

## 引用

```example
「おはよう」と言う。#说「早上好」
```
