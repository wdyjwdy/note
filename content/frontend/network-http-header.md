## Content-Type

Used to indicate the media type of a resource.

- `text/plain`
- `text/html`
- `text/javascript`
- `text/css`
- `application/json`
- `application/xml`
- `multipart/form-data`
- `application/x-www-form-urlencoded`
- `image/png`
- `image/jpeg`
- `image/svg+xml`
- `font/woff2`
