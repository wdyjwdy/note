## 即使

{% examples %}
小学生でもわかる。#小学生都知道。
１円でも安いほうがいい。#即使是1日元，也是便宜的更好。
{% endexamples %}

## 列举

通常在表示提议的句子中使用。

{% examples %}
お酒でも飲みませんか。#喝点酒什么的吗。
コーヒーでも飲みましょう。#喝点咖啡什么的吧。
{% endexamples %}

## 都

{% forms %}
〔疑问词〕でも
{% endforms %}

{% examples %}
誰でも行ける。#谁都可以去。
花はどこのでもある。#花哪里都有。
{% endexamples %}
