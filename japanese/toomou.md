## 觉得

{% forms %}
〔简体〕と思う
{% endforms %}

觉得。

```example
とてもよかったと思います。#我觉得很好。
```

{% forms %}
〔简体〕と思っている
{% endforms %}

一直这样觉得。第三人称使用「と思っている」。

```example
彼は私が日本にいると思っています。#他一直觉得我在日本。
```

## 想

{% forms %}
〔意向形〕と思う
{% endforms %}

想。

```example
出かけようと思います。#想出门。
```

{% forms %}
〔意向形〕と思っている
{% endforms %}

一直有某个想法。第三人称使用「と思っている」。

```example
車を買おうと思っています。#一直想买车。
```

{% forms %}
〔意向形〕とは思いません
{% endforms %}

不想。「と思う」的否定。

```example
一人で行こうとは思いません。#不想一个人去。
```
