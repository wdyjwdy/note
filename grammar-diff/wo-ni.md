---
title: を・に
---

## 场所

- **を**：离开的场所。
- **に**：进入的场所。

{% examples %}
教室に入る。#进入教室。
教室を出る。#离开教室。
電車に乗る。#进入电车。
電車を降りる。#离开电车。
日本に行く。#进入日本。
日本を出る。#离开日本。
{% endexamples %}

## 动作

- **を**：动作作用的对象。
- **に**：动作接收的对象。

{% examples %}
魚を食べる。#吃鱼。
水を飲む。#喝水。
恋人に花をあげる。#给恋人花。
友達に傘を借りる。#向朋友借伞。
{% endexamples %}
