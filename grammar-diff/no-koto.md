## 名词化

- 「[の](/grammar-list/no#名词化)」：名词化。
- 「[こと](/grammar-list/koto#名词化)」：名词化。

使用「こと」的场景。

{% examples %}
日本語を話すことができます。#能力
趣味は写真を撮ることです。#句末（「のです」会与「んです」发生歧义）
{% endexamples %}

使用「の」的场景。

{% examples %}
鳥が鳴いているのが聞こえる。#听觉
人が歩いているのが見える。#视觉
探しているのは牛乳だ。#强调
{% endexamples %}
