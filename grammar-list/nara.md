title: なら

## 表示假设

{% forms %}
〔なら形〕
{% endforms %}

如果～。同义词[ば](./ba)。不能接意志和劝诱。

{% examples %}
君なら、きっとできる。#是你的话，一定能做到。
{% endexamples %}

## 表示判断

{% forms %}
〔简体・な形・名词〕
{% endforms %}

～的话。可以接意志和劝诱。

{% examples %}
ケーキなら、駅前の店がおいしいですよ。#蛋糕的话，车站前那家店很好吃。
テレビ見ていないなら、消す。#电视不看话，就关了。
{% endexamples %}
