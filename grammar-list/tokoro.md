---
title: ところだ
---

## 即将做

{% forms %}
〔字典形〕ところだ
{% endforms %}

{% examples %}
試合が始まるところです。#比赛就要开始了。
今帰るところです。#现在正要回家。
{% endexamples %}

## 正在做

{% forms %}
〔ている〕ところだ
{% endforms %}

{% examples %}
牛乳を飲んでいるところです。#正在喝牛奶。
{% endexamples %}

## 刚做完

{% forms %}
〔た形〕ところだ
{% endforms %}

{% examples %}
ご飯を食べたところです。#刚吃完饭。
{% endexamples %}
