---
title: ことがある
---

## 以前做过某事

{% forms %}
〔た形〕ことがある
{% endforms %}

{% examples %}
日本へ行ったことがあります。#去过日本。
一度も海を見たことがない。#一次海都没看过。
{% endexamples %}

## 偶尔发生某事

{% forms %}
〔字典形・ない形〕ことがある
{% endforms %}

助词有时会用「も」。

{% examples %}
時々紙の辞書を使うことがある。#偶尔使用纸质词典。
時間がある時は自分で料理を作ることもある。#有时间的时候，也会自己做饭。
{% endexamples %}
