## 请求解释

{% forms %}
〔简体・名词な・な形な〕んです・のです
{% endforms %}

{% examples %}
どこへ行くんですか。#去哪里？
どうして遅れたんですか。#为什么迟到？
どこへ行くの？#去哪里？
どうして遅れたの？#为什么迟到？
{% endexamples %}

## 表示原因

{% forms %}
〔简体・名词な・な形な〕んです・んだ
{% endforms %}

{% examples %}
熱があるんです。#为什么请假？因为发烧了。
熱があるんだ。#为什么请假？因为发烧了。
{% endexamples %}
